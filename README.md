# Vulkan RAII Hello Compute

Vulkan Hello Compute using `vk::raii`

## Clone

```
git clone --recursive https://github.com/yknishidate/Vulkan-RAII-Hello-Compute.git
cd Vulkan-RAII-Hello-Compute
mkdir build
cd build
```

## Build & Run

```
// windows
cmake ..
cmake --build .
.\Debug\vulkan_compute.exe
```
